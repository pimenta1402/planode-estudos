## 汳｡ DICAS IMPORTANTES

### **Question Tags:**
- Se a frase ﾃｩ positiva, a tag ﾃｩ negativa
- Se a frase ﾃｩ negativa, a tag ﾃｩ positiva
- Use o mesmo auxiliar da frase principal
- Para "I am", use "aren't I"

### **First Conditional:**
- If + present simple, will + infinitive
- Use para situaﾃｧﾃｵes reais e possﾃｭveis
- Pode usar "unless" no lugar de "if not"
